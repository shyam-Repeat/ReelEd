# Android Expert Skill

The `android-expert` skill is a comprehensive toolkit for Android developers. it integrates advanced Jetpack Compose architectural patterns with robust UI verification techniques using ADB.

## Features

- **Jetpack Compose Mastery**: Guidance on state management (MVI/MVVM), optimized recomposition, and type-safe navigation.
- **Automated UI Verification**: Systematic workflow for interacting with and verifying app state on an emulator using ADB and `uiautomator`.
- **Performance & Debugging**: Best practices for stable UI states and fast troubleshooting of layout issues.

## Usage

Trigger this skill when:
- Designing or implementing new Jetpack Compose screens.
- Setting up navigation or state flows.
- Verifying UI behavior autonomously.
- Debugging emulator-specific interaction bugs.

## Installation

This skill is stored locally in `.gemini/skills/android-expert`. To use it, ensure it's activated or in the search path of your Gemini CLI.
